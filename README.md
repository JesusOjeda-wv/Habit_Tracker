# Habit_Tracker
Serverless Habit Tracker app built on AWS. Create habits, check in on them, and track your streaks.

**Status: In development.**

## Overview

Habit Tracker is a small serverless application. It allows you to create habits to keep track of, check on, and build up a large streak. Currently in progress.

## Tech Stack
- Java 21
- React/TS
- S3 + CloudFront
- API Gateway
- AWS Lambda
- Dynamo DB
